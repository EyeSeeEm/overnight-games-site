# Implemented Features: System Shock 2D (Canvas)

## Core Mechanics
- [x] WASD 8-directional movement
- [x] Mouse aim (player rotates toward cursor)
- [x] Twin-stick style shooting
- [x] Flashlight cone visibility system
- [x] Raycasting vision (stops at walls)
- [x] Collision detection with walls

## Combat System
- [x] Left click to fire weapon
- [x] Reload mechanic (R key)
- [x] Multiple weapon types (melee, pistol, shotgun, SMG, laser)
- [x] Critical hit system (15% chance, 2x damage)
- [x] Floating damage numbers
- [x] Kill streak system with XP bonuses
- [x] Dodge roll with i-frames (Space)
- [x] Screen shake on damage
- [x] Damage flash effect

## Movement Mechanics
- [x] Sprint (Shift, costs energy)
- [x] Crouch (Ctrl, quieter, slower)
- [x] Dodge roll (Space, 15 energy cost)
- [x] Collision sliding along walls
