# Implemented Features: system-shock-2d (littlejs)

## Core Mechanics
- [x] Twin-stick controls (WASD + mouse aim)
- [x] Player rotation follows mouse
- [x] Sprint system (Shift key, costs energy)
- [x] Crouch system (Ctrl key, slower, harder to detect)
- [x] Dodge roll (Space, i-frames, energy cost)
